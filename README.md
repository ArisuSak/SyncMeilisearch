# NATS JetStream Integration

This project demonstrates how to integrate NATS JetStream with PostgreSQL and Meilisearch using Go.

## Prerequisites

- Go 1.16 or later
- PostgreSQL
- NATS Server
- Meilisearch

## Run application
```sh
cd nats-jetstream
```
```sh
go run cmd/main.go
```

# Install dependencies:
go mod tidy
